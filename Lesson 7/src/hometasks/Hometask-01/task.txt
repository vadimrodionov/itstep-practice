Создать текстовый редактор "mote" (My Own Text Editor).

---======---

Функции: 
- создание нового файла
- открытие существующего файла
- редактирование содержимого файла
- сохранение файла
- сохранение файла с произвольным именем
- вывод справки о редакторе

=== Запуск редактора ===

Запуск редактора осуществляется из интерфейса командной строки 
с помощью команды "mote".

Варианты запуска: 
1. `mote` - запуск редактора без создания файла
   
   При данном варианте запуска редактор открывается без содержимого, а 
   введённый пользователем текст будет храниться в оперативной памяти до 
   завершения работы редактора.
   При закрытии редактор должен спросить пользователя, хочет ли он сохранить 
   введённый текст. Если пользователь отвечает "Да", редактор должен создать 
   новый файл.

2. `mote -o path/to/file.txt` - запуск редактора с открытием существующего 
   файла по указанному пути
   
   При данном варианте запуска редактор должен открыть файл по указанному 
   пути. Если файл недоступен или не существует, редактор должен вывести 
   сообщение об ошибке доступа с указанием причины и завершить работу.

3. `mote -n patn/to/new-file.txt` - запуск редактора с созданием нового файла
   по указанному пути

   При данном варианте запуска редактор создаёт новый файл по указанному пути 
   и открывает его. Если файл не удалось создать, или по указанному пути в
   системе уже существует файл, редактор должен вывести сообщение об ошибке 
   создания файла с указанием причины и завершить работу.

4. `mote -h` - вывод справки
   
   При данном варианте запуска редактор выводит справку о себе и завершает 
   работу.
   Справка должна содержать:
   - название редактора,
   - описание вариантов запуска редактора,
   - описание возможностей редактора по работе с текстом и руководство по 
     использованию этих возможностей,
   - описание возможностей по сохранению текста и руководство по 
     использованию этих возможностей.

=== Работа с текстом ===

Редактор должен содержать следующие возможности работы с текстом:
- навигация (перемещение курсора) в пределах одной строки,
- навигация между строками с сохранением положения курсора в пределах строки,
- удаление символа перед курсором,
- удаление символа после курсора,
- вставка символа перед курсором.

Навигация в пределах одной строки осуществляется с помощью клавиш "LeftArrow" 
и "RightArrow".
Навигация между строками осуществляется с помощью клавиш "UpArrow" и 
"DownArrow".
Удаление символа перед курсором осуществляется с помощью клавиши "Backspace".
Удаление символа перед курсором осуществляется с помощью клавиши "Delete".
Вставка символа осуществляется при нажатии любой символьной клавиши.

=== Сохранение текста в файл ===

Редактор должен поддерживать два варианта сохранения текста: сохранение в 
новый файл и сохранение в открытый файл.

Если редактор был запущен без открытия или создания файла, доступна только 
опция сохранения в новый файл.

Если редактор был запущен с открытием или созданием файлов, по умолчанию 
сохранение текста должно производиться в открытый файл.

Опция сохранения в открытый файл должна быть доступна по нажатию клавиш 
"Ctrl+S", опция сохранения в новый файл должна быть доступна по нажатию 
клавиш "Ctrl+Shift+S". 
Опция выхода без сохранения должна быть доступна по нажатию клавиш "Ctrl+W".
